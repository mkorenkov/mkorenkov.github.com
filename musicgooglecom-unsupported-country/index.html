music.google.com Unsupported Country

Dec 25th, 2011

UPD. Статья на engadget о Tor + music.google.com: http://www.engadget.com/2011/11/19/how-to-use-google-music-from-anywhere-yes-outside-of-the-unit/

Вчера сидели у Толика, слушали музыку c music.google.com. Попытавшись зарегистрировать для себя, получил отказ - Unsupported country. Все для страны победившего капитализма, значит. Решим эту проблему на примере Ubuntu.

Потыкавшись в разные анонимные web proxy, выяснилось, что они не поддерживают javascript. Рабочую proxy на территории США найти с пол-пинка не удалось. На помощь пришел Tor сразу после победы над ледяными гигантами. Проект Тор - это сеть виртуальных туннелей по всему миру для повышения анонимности тех, кому нужно.

Итак, установим себе Tor и Vidalia - GUI к нему:

sudo apt-get install tor tor-geoipdb vidalia

Через System Preferences \ Network настраиваем proxy-сервер.

127.0.0.1:9050

Запускаем Vidalia, ищем хосты в США (Browse Network). Во вкладке Advanced редактируем torrc, заполняя реальными данными:

ExitNodes server1, server2, server3
StrictExitNodes 1

Перезапускаем tor и идем регистрироваться на music.google.com.

Profit!
