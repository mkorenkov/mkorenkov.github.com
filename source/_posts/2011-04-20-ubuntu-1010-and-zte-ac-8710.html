Переодически я настраиваю PeopleNet для Ubuntu. Для версий, которые младше 10.10, я пользовался старым постом onorua. Сегодня, после привычной modprobe (ниже) Network manager подпрыгнул и сказал, что готов настроить для меня Mobile broadband связь.

modprobe usbserial vendor=0x19d2 product=0xfffe

В списке уже был PeopleNet, что давало определенную надежду. Однако из коробки он не завелся. В процессе я опробовал CrossPlatformUI-V1.0.27-BSNL-i386-ubuntu и несколько менее экзотических способов.

В финале, ZTE 8710 и Ubuntu удалось скрестить путем установки нового Network Manager из ppa (https://launchpad.net/~network-manager/+archive/ppa).

Я по-старинке прописал в /etc/apt/sources.list:

 

deb http://ppa.launchpad.net/network-manager/ppa/ubuntu maverick main 

deb-src http://ppa.launchpad.net/network-manager/ppa/ubuntu maverick main

Выполнил:

 

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys BC8EBFE8

sudo aptitude update

sudo aptitude upgrade

И вуаля, Network Manager в Ubuntu 10.10 теперь может использовать мой 3g-модем.

Осталось прописать modprobe usbserial vendor=0x19d2 product=0xfffe в /etc/rc.local, чтобы после перезагрузки все продолжало работать.
