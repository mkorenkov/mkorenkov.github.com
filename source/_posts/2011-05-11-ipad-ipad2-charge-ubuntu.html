It appears the problem with Ubuntu and iPad / iPad2 USB charging is bringing attention. Thus, I would write about charging iPad and iPad2 under Ubuntu. This solution works well for Ubuntu 10.10 (Maverick Meerkat) and Ubuntu 11.04 (Natty Narwhal).

There’s a program created by Ondrej Zary from Rainbow-Software. It had some flaws, which were fixed here: https://github.com/mkorenkov/ipad_charge/

Installation for iPad

sudo apt-get install libusb-1.0-0 libusb-1.0-0-dev
wget --no-check-certificate https://github.com/downloads/mkorenkov/ipad_charge/ipad_charge_1.0.tar.gz
tar -xzf ipad_charge_1.0.tar.gz
cd ./ipad_charge_1.0
make
sudo make install

 

Installation for iPad2

sudo apt-get install libusb-1.0-0 libusb-1.0-0-dev
wget --no-check-certificate https://github.com/downloads/mkorenkov/ipad_charge/ipad2_charge_1.0.tar.gz
tar -xzf ipad2_charge_1.0.tar.gz
cd ./ipad2_charge/
make
sudo make install
