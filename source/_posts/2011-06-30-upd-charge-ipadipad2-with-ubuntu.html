For those, who are interested.

There was a patch by dirigeant that allows you charge both iPad and iPad2 with a single Ubuntu computer.

[[posterous-content:CEixbgfvbEDrlxduHwoA]]

Installation instructions:

sudo apt-get install libusb-1.0-0 libusb-1.0-0-dev
wget --no-check-certificate https://github.com/downloads/mkorenkov/ipad_charge/ipad_charge.tar.gz
tar -xzf ipad_charge.tar.gz
cd ./ipad_charge
make
sudo make install
