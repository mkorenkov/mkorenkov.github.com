Hudson

Hudson is a continuous integration (CI) tool written in Java, which runs in a servlet container, such as Apache Tomcat or the GlassFish application server. It supports SCM tools including CVS, Subversion, Git and Clearcase and can execute Apache Ant and Apache Maven based projects, as well as arbitrary shell scripts and Windows batch commands.

Unfortunately, you can try it difficult to use with Google App Engine projects.

Installing Hudson

Installing Hudson is easy for major Linux distributives:

  • Debian/Ubuntu
  • Fedora/CentOS/etc

For more information, please visit Installing Hudson

Configuring Hudson

Press New Job (Build a free-style software project)

Source Code Management

  1. Copy read-only repository URL (from GitHub)
  2. Select Source Code Management system (e.g. GIT, paste URL)
  3. Press Advanced
  4. Select Skip internal tag

Add Build Step

  1. Press Add Build Step
  2. Select Execute shell
  3. Add something like the following code
    echo GOOGLE_PASSWORD | python2.6 /usr/local/google_appengine/appcfg.py --email=GOOGLE_EMAIL --passin update $WORKSPACE
