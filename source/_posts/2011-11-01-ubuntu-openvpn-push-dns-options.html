--- layout: post title: Ubuntu openvpn push DNS options published: true ---

По умолчанию, Ubuntu не просасывает DNS опции через openvpn. В логах прекрасно видно, что опции приезжают, но /etc/resolv.conf при этом не обновляется. Исправляется так:

sudo aptitude install resolvconf
sudo mv /etc/resolv.conf /etc/resolv.conf.orig
sudo ln -s /etc/resolvconf/run/resolv.conf /etc/resolv.conf

Теперь в конфигурационный файл клиента нужно добавить 2 строчки:

up /etc/openvpn/update-resolv-conf
down /etc/openvpn/update-resolv-conf

Оригинал доступен здесь: http://plug.4aero.com/Members/lmarzke/howto/openvpn-push-dns
