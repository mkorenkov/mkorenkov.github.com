--- layout: post title: Disabling Enhanced Internet Security in Windows 2008 Server published: true ---

Пришлось поконфигурить Microsoft Windows 2008 Server. Я молчу про то, что до сих пор приезжает Ehnanced Internet Security и про то, какой уебан ее придумал.

Отключить ее можно так: Открыть Server Manager, найти Configure IE ESC, кликнуть, отключить к ебенятам. Простите, скриншотиков не будет.
